| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-4-07 | Information Disclosure | CLI outputs from core functions | mitigate | Core functions apply input sanitization (sanitizeMessageId, folderPathToFilename) before file access; CLI only outputs what core returns |

| T-4-10 | Tampering | .gitignore modification during checkout | mitigate | .gitignore entry is safe (only adds '.worktrees/' line); no path traversal possible in entry content |
</threat_model>
